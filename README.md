![Opening page](https://github.com/user-attachments/assets/6701ff5e-7449-47db-bb9b-28edfa24e6c7)### MILITARY-GRADE PORTFOLIO
A cybersecurity-themed interactive portfolio with terminal interface and biometric scanner animation.

## Tech Stack
HTML5 - Structure

CSS3 - Styling with custom animations

JavaScript - Terminal logic & effects

Three.js - 3D elements

## Features
Biometric scanner animation overlay

Interactive terminal with command system

Matrix rain effect (easter egg)

Fully responsive design

Military aesthetic UI

## Available Commands
Command	Description
help	Display all commands
about	View operative dossier
mission	Active mission parameters
skills	Technical capabilities
contact	Request contact protocols
clear	Clear terminal screen

## Quick Start
bash
# Clone and open
git clone <repo-url>
cd military-portfolio
# Just open index.html in browser
